Краткое описание новых возможностей

  1. Search - при переходе отображает все посты без фильтров. Поиск осуществляется, как по каждому в отдельности, так и по нескольким фильтрам.
  2. Create - страница выбора опредленного типа поста. После выбора переносит на создание поста.
  3. Кнопки редактирования - при нажатии открывают выбранный пост в режиме редактирования. Тамже можно удалить пост.
